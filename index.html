Study Cards

Red/blue circular sign

No parking/no stopping. Broad Europe-style clue, not a country lock. Check language, plates, bollards, and road numbers.

Stop words

DUR = Turkey. ALTO = Mexico/Latin America. PARE = Brazil/Latin America. STOP alone is weak.

Yellow outer shoulder

Left-driving plus yellow outer shoulder line strongly points to South Africa region.

Scripts

Hangul = Korea. Kana = Japan. Greek = Greece. Hebrew = Israel. Thai = Thailand. Khmer = Cambodia. Lao = Laos.

Plates

Blue strip = Europe-style. Yellow rear + left driving = UK likely. Yellow both plates could be Netherlands, Luxembourg, Colombia, Israel, etc.

Language separators

Rua/Estrada/ção/nh/lh = Portuguese. Calle/Carretera/ñ = Spanish. Katu/tie/järvi = Finland. Kabupaten/Kecamatan = Indonesia.

Landscape traps

Red dirt, palm trees, mountains, and vineyards are broad clues. They need signs/language/roads to become useful.

Master rule

Never lock from one clue unless it is extremely strong. Stack clues: sign + language + road + plate + landscape.

Real image status

Every question now searches Wikimedia Commons for a reusable real image using the clue type. If no image loads, the game clearly labels and shows the illustrated fallback.
